# 🐳 Dockerfile Generator

> Generate production-ready Dockerfiles visually — no YAML knowledge needed.

🔗 **Live demo:** [meriemcnsendjek.github.io/dockerfile-generator](https://meriemcnsendjek.github.io/dockerfile-generator/)

![Dockerfile Generator Screenshot](./screenshot.png)

## ✨ Features

- 🎯 Visual configuration — no manual Dockerfile writing
- 🐘 Supports PHP/Laravel, Node.js, Python, and Java/Spring
- ⚡ Real-time preview as you configure
- 📋 One-click copy to clipboard
- 📥 Download Dockerfile instantly
- 🌙 Dark mode
- 🔧 Custom environment variables
- 💯 100% free, no backend required

## 🚀 Why I built this

Writing `Dockerfile`s manually is repetitive and error-prone, especially for developers new to Docker. This tool lets you pick your stack, configure a few options, and get a production-ready Dockerfile in seconds.

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript
- **Styling:** TailwindCSS v4
- **Icons:** Lucide React
- **Build tool:** Vite
- **Deployment:** GitHub Pages

## 📦 Getting Started Locally

```bash
# Clone the repository
git clone https://github.com/meriemcnsendjek/dockerfile-generator.git

# Navigate to the project
cd dockerfile-generator

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open `http://localhost:5173` in your browser.
